# Atividades - Programação de Aplicativos Mobile II 📱

Este repositório armazena os projetos e atividades propostas durante as aulas de **Programação de Aplicativos Mobile II**. O foco principal é o desenvolvimento com **React Native** e a exploração de bibliotecas do ecossistema Expo.

## 🚀 Tecnologias Utilizadas

* [React Native](https://reactnative.dev/)
* [Expo](https://expo.dev/)
* [React Navigation (Drawer)](https://reactnavigation.org/docs/drawer-navigator/)
* [Expo Vector Icons](https://icons.expo.fyi/)

## 🛠️ Funcionalidades

- **Drawer Navigator**: Implementação de menu lateral para transição fluida entre os diferentes componentes e atividades do app.
- **Home Personalizada**: Uma tela inicial que apresenta minhas informações e links para redes sociais (GitHub, LinkedIn e E-mail).
- **Componentização**: Organização do código focada em reutilização e Clean Code.

## 📦 Instalação e Execução

Para rodar este projeto localmente, siga os passos abaixo:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/najumattos/AtividadeReactNative/

2. **Instalar dependências:**
   ```bash
   npm install

3. **Drawer Navigation:**
   ```bash
   npx expo install @react-navigation/drawer

4. **Executar o Projeto:**
   ```bash
   npx expo start
